# **App Name**: FOCI Connect

## Core Features:

- Company Information Display: Display FOCI Group's contact details, physical address, and website information as provided in the company document header. The company logo is rendered on the page, and positioned to match the document header.
- Document Information Fields: Provide editable fields for Document Number, Date, Revision, and Page Number, pre-populated with default or last-entered values.
- AI-Powered Content Summarization: Use a generative AI tool to automatically summarize documents or articles based on their content to display as a preview. 
- Contact Form Submission: Enable users to submit inquiries via a contact form that sends data to FOCI Group's email (info@foci.group).
- Service Showcase: Visually showcase the core services of FOCI Group: Engineering, ICT, and Smart Automation.

## Style Guidelines:

- Primary color: Dark slate blue (#4361EE), conveying professionalism and technical expertise.
- Background color: Light gray (#E5E7EB), provides a neutral backdrop for content.
- Accent color: Vibrant orange (#FA7000) to highlight key interactive elements and calls to action.
- Headline font: 'Space Grotesk', a geometric sans-serif for a modern and technical feel.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use modern, outline-style icons to represent services and functionalities.
- Maintain a clean, grid-based layout to ensure readability and visual structure, similar to the provided document header.
